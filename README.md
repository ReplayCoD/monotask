# Monotask

> Eine ADHS-freundliche Fokus-App, die immer nur eine Aufgabe zeigt – die wichtigste für genau jetzt.

## Das Problem
Klassische To-Do-Apps überfordern mit endlosen Listen. Monotask zeigt bewusst nur einen Task.

## Features
- Automatische Priorisierung nach Deadline und Dauer
- Minimalistisches Interface – kein visuelles Rauschen
- Lokale H2-Datenbank – deine Daten bleiben bei dir

## Tech Stack
- Java 17
- JavaFX 21
- H2 Database (embedded)
- Maven
